# Weather-application

A simple and responsive Weather Forecast Web App built using HTML, CSS, JavaScript, PHP, and MySQL, integrated with the WeatherAPI to provide real-time weather updates and forecasts

Features:

Search weather by city name
Real-time current weather data
7 day weather forecast and 7 days historical weather data display
Temperature, humidity, wind speed, and condition display
earch history storage using MySQL
Fast and API-based data fetching (WeatherAPI)
Responsive design for mobile and desktop

Frontend:
HTML5
CSS3
JavaScript (AJAX / Fetch API)

Backend:
PHP
JavaScript

Database:
MySQL
API
WeatherAPI
